# Brazilian E-Commerce Sales Analysis 🇧🇷
**From SQL Exploration to Tableau Dashboard**

## 1. Project Background
Project ini menganalisis dataset publik Olist (e-commerce Brasil) periode 2016-2018. 
- **Dataset Size:** 99.441 customers, 112.650 order items, & 32.951 products.
- **Objective:** Memahami pola penjualan, perilaku pelanggan, dan performa produk untuk pengambilan keputusan bisnis.

## 2. Technical Stack
- **Database:** SQL (PostgreSQL/MySQL) for Data Extraction & Agregation.
- **Visualization:** Tableau for Interactive Dashboard.
- **Concepts:** ERD, Pareto Pattern, Seasonality Analysis.

## 3. Key SQL Analysis Results
Berdasarkan eksplorasi SQL (cek file `.sql` di repo ini):
- **Total Revenue:** 13.591.643,70 BRL.
- **Seasonality:** Penjualan memuncak secara konsisten pada bulan **November** (efek Black Friday).
- **Urban Centralization:** Pendapatan terbesar terpusat di wilayah urban seperti **São Paulo**.
- **Product Leader:** Kategori `cama_mesa_banho` menjadi kategori terlaris.

## 5. Strategic Recommendations
- **Retention Program:** Fokus pada program loyalitas untuk segmen *High-Value Customer*.
- **Peak Season Readiness:** Optimalisasi stok 2 bulan sebelum November (Black Friday).
- **Product Prioritization:** Alokasi budget promosi lebih besar pada kategori *Best-Seller*.
